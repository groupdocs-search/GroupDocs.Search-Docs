# GroupDocs.Search - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Search](https://docs.groupdocs.com/search/) static documentation.

## About GroupDocs.Search

[GroupDocs.Search](https://products.groupdocs.com/search/) is a document search SDK that enables you to perform full-text search across documents with advanced indexing, faceted search, and query capabilities.

Available for .NET, Java, and Node.js.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/search/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
